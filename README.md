# AI-Powered E-Commerce Chatbot

A clean, modern AI assistant built to handle automated customer sales and support with zero delay. Instead of just answering basic questions, this chatbot acts as a virtual sales agent—handling order placements, displaying product images dynamically, and even negotiating prices with buyers in real-time. 

## ✨ Features
* **Smart Negotiation:** Dynamically handles price bargaining and counter-offers to help seal the deal.
* **Order Placement:** Automates the checkout process by capturing user intent and taking orders directly inside the chat.
* **Visual Product Showcase:** Delivers high-quality product images to users based on their requests.
* **Instant Messaging & Streaming:** Ultra-fast responses powered by Groq and OpenRouter with zero lag.
* **Smart Auto-Scroll:** Automatically scrolls down to the newest message to keep the conversation flowing.
* **Live Typing Indicator:** Shows a fluid loading animation while the AI fetches real-time data.
* **Fully Responsive:** Beautifully optimized for both mobile screens and desktop layouts.

## 🛠️ Tech Stack
* **Framework:** Next.js
* **Styling:** Tailwind CSS
* **LLM APIs:** Groq, OpenRouter
